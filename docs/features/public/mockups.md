# Mockups (Single Challenge)

The mockups page is a development and design reference for the single challenge landing page layout. It's not a production feature — it exists to preview and iterate on the challenge page design.

## What It Shows

A static mockup of the challenge detail page with placeholder content. Useful for:
- Previewing the layout and visual design.
- Testing responsive behavior across screen sizes.
- Iterating on the hero section, info cards, enrollment CTA, and content sections.

## Access

The mockup is accessible at `/mockups/single-challenge` and is only useful during development.
